================================================
PROJECT: SIMPLE COUNTER DOCUMENTATION
LANGUAGE: HTML / JAVASCRIPT
================================================

1. OVERVIEW
-----------
This code creates a basic web-based counter. It provides 
a visual display of a number and three buttons to 
manipulate that number: Subtract, Reset, and Add.

2. COMPONENT BREAKDOWN
----------------------
A. DISPLAY ELEMENT
   - An <h1> tag with id="s".
   - This serves as the "screen" for the counter.

B. DATA INITIALIZATION
   - variable 'a': Stores the numeric value (starts at 0).
   - variable 's': A reference to the HTML display element.

C. FUNCTIONAL BUTTONS
   - SUBTRACT: Decrements the value of 'a' by 1 and 
     updates the display.
   - RESET: Sets the value of 'a' back to 0 and 
     updates the display.
   - ADD: Increments the value of 'a' by 1 and 
     updates the display.

3. EXECUTION FLOW
-----------------
1. The browser renders the <h1> and the three buttons.
2. The script initializes the counter variable to 0.
3. When a user clicks a button:
   a. The JavaScript math logic runs (e.g., a = a + 1).
   b. The script targets the <h1> element by its ID ("s").
   c. The text inside the <h1> is replaced by the new 
      value of 'a'.

4. NOTES
--------
- The logic is "inline," meaning the JavaScript functions 
  are written directly inside the HTML button tags.
- The state is lost if the page is refreshed, as the 
  variable 'a' resets to 0 upon loading.
================================================
